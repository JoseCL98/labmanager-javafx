
# Gestor de Inventario y Préstamos - Java/JavaFX (Lab Manager)

## Resumen
Proyecto educativo que implementa un gestor de inventario con préstamos en Java 17 + JavaFX.
Incluye patrones SOLID, GoF y GRASP, pruebas, documentación y un UI básico en JavaFX.

## Requisitos
- Java 17+
- Maven 3.8+
- Para ejecutar con JavaFX use el plugin javafx-maven-plugin

## Ejecutar (maven)
En la raíz del proyecto (donde está pom.xml):
```
mvn javafx:run
```

## Qué incluye
- Código fuente en `src/main/java`
- Recursos FXML en `src/main/resources/main.fxml`
- Implementaciones en memoria (fáciles de sustituir por JDBC/SQLite)
- Documentación en `docs/`
